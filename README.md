# Rebirth Phoenixfest Website

The official launch site for Rebirth Phoenixfest—a new gathering of sound, art, movement, and collective renewal.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production checks

```bash
npm run lint
npm run build
```

Built with Next.js, React, and TypeScript. Designed for deployment on Vercel.
